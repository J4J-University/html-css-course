Paragraphs (<p>)

Paragraphs are the most basic element in HTML. They are used to organize the content of a page.

Paragraphs are separated from each other by a blank line. In this document, each section is a paragraph.

Preformatted text (<pre>)

Preformatted text is used to display text that preserves both spaces and line breaks.

Here is an example of preformatted text:

                This is a preformatted text.
                It preserves both spaces and line breaks.

Preformatted text is useful to display code, for example:

                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Preformatted text example</title>
                    </head>
                    <body>
                        <h1>Some heading bla bla bla</h1>
                    </body>
                </html>

Lists (<ul>, <ol>, <li>)

Lists are used to display a list of items. There are two types of lists: ordered lists and unordered lists.

Here is an example of an ordered list:

  1. First item
  2. Second item
  3. Third item

Here is an example of an unordered list:

  • First item
  • Second item
  • Third item

You can also create nested lists:

  • First item
  • Second item
  • Third item
  • Fourth item
    • First item
    • Second item
    • Third item

You can also create a list of links. To do so, you need to use the <a> tag inside the <li> tag.

  • W3Schools
  • Google
  • YouTube

You can also create a list of buttons. To do so, you need to use the <button> tag inside the <li> tag.

Generic containers (<div>, <span>, <section>)

The <div> tag defines a division or a section in an HTML document.

The <span> tag is an inline container used to mark up a part of a text, or a part of a document.

The <section> tag defines sections in a document, like chapters, headers, footers, or any other sections of the document.
