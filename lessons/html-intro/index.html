Lesson 1 - HTML Intro

Intro to HTML

What is HTML?

HTML זוהי שפה שמגדירה את המבנה והתוכן של דפי אינטרנט.

ראשי התיבות של HTML הן: HyperText Markup Language.

HTML Syntax

אבני הבניין של HTML הם אלמנטים. אלמנטים יכולים להגדיר פיסות ממשק קטנות, מבנה של חלק מהדף וגם לתת מידע על הדף.

בשביל להגדיר אלמנטים נשתמש בתגיות. התגיות מייצגות את הסוג של האלמנט, ויכולות לספק עוד מידע על האלמנט.

התחביר של הגדרת אלמנט הולך ככה:

יש תגית פותחת , אחריה יש את ה- תוכן של האלמנט, ואז תגית סוגרת .

המונח ״אלמנט״ מתייחס למכלול של התגית הפותחת, התוכן והתגית הסוגרת.

התוכן יכול להיות טקסט, או אלמנטים נוספים של HTML. למעשה דף HTML מורכב כולו מ- "עץ" של אלמנטים.

Good HTML vs. Bad HTML

יש הבדל גדול בין לכתוב HTML שיעבוד לבין לכתוב HTML טוב.

כדי לכתוב HTML טוב צריך להכיר את הייעוד של כל אלמנט ולהשתמש בו בהתאם.

אפשר לכתוב קובץ HTML שנראה כמו תאונת דרכים אבל עדיין הדפדפן יצליח להבין אותו איכשהו. אנחנו צריכים לשאוף לכתוב קבצי HTML באופן כזה שמתי שאדם אחר מסתכל עליו הוא יצליח להבין מהו המבנה של הדף כמה שיותר בקלות.
