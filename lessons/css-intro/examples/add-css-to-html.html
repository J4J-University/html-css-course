Inline CSS

To add CSS to an HTML element, you can use the style attribute.

For example, to make the text of this paragraph red, you can use the style attribute and set the value to color: red.

A red paragraph

Internal CSS

To add CSS to an HTML document, you can use the <style> element.

For example, to make the text in the <span> blue, you can use the <style> element and set the value to color: blue. I'm a blue span!

External CSS

To add CSS to an HTML document, you can use an external CSS file.

For example, to make the text in the <span> green, you can use an external CSS file and set the value to color: green.

I'm a green span!

@import

You can use the @import rule to import an external CSS file into another CSS file.

For example, we define a salmon color to a class in the stylesheet named sheet2.css and then we import it in the stylesheet named sheet1.css.

And now we can use the salmon color in the HTML document
